Les CSS représentent une nouvelle façon d'appliquer des styles aux éléments HTML.Ils vous permettent de définir n'importe quelle propriété de style comme la bordure, la couleur de fond, le type de caractère, l'espace entre les lettres, etc.Il y a trois façons de définir des styles CSS :
*Dans le corps du code HTML        *Dans l'en-tête de la page      *Dans une feuille de style totalement séparée du code HTML
